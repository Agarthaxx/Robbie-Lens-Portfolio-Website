## Robbie Lens | Portfolio Website 

Robbie Lens is a modern and responsive portfolio website build with HTML and CSS as part of the OpenClassrooms web development course.  
The project showcase a photographer's work through a clean design, structured layout and optimized user experience.  

## 🚀 Features  
	•	Responsive design (desktop, tablet, mobile)  
	•	Clean and semantic HTML structure  
	•	Modern CSS styling (Flexbox, layout techniques)  
	•	Image gallery showcasing photography work  
	•	Navigation between multiple pages  
	•	Accessible and user-friendly interface  

## 🛠️ Technologies Used  
	•	HTML5  
	•	CSS3 (Flexbox)  
	•	Google Fonts  
	•	Basic SEO and accessibility practices  

## 📂 Project Structure
	•	index.html – Homepage
	•	about.html – About page
	•	portfolio.html – Gallery of projects/photos
	•	css/ – Stylesheets
	•	images/ – Assets and media  

## 🎯 Purpose
    This project was created as part of the OpenClassrooms curriculum to practice:  
	•	Structuring a website with semantic HTML  
	•	Styling with CSS and responsive design principles  
	•	Building a real-world portfolio layout  
	•	Understanding UI/UX basics  

## 📸 Preview  
    The website represents the portfolio of a fictional photographer, Robbie Lens, highlighting visual storytelling and clean presentation.
